1. **Introduction** 


|<p>**1.1 개요**	</p><p>- 2022-2 프로그래밍응용 프로젝트를 위해 파이썬으로 웹사이트를 개발하였다.</p><p>- 해당 프로젝트는 웹프레임워크인 Flask와 현재 베타 개발 단계인 PyScript를 이용하여 개발하였다.</p><p>- 웹사이트를 개발할 때 필수요소 중 하나인 js(javascript)를 대체할 언어로 파이썬에서 개발한 PyScript가 </p><p>발표되었다는 소식에 해당언어를 공부하고 싶었고 마침 오요셉 교수님의 프로그래밍응용 수업에서 파이썬을 이용한 프로젝트를 진행해야 했기에 위와 같은 주제를 선택하게 되었다.</p><p></p><p>![ref1]![ref2]</p><p>![ref3]![ref4]</p><p>**1.2 추진배경**</p><p>- 날로 높아지고 있는 파이썬의 사용비율과, 넓은 활용분야 덕분에 파이썬은 모든 공대생의 필수 프로그래밍 언어가 되어 보다 높은 이해도와 공부의 필요성을 느꼈다..</p><p>- 파이썬에서 2022에 발표한 PyScript(현재Beta버젼) 또한 빠른 시일 내로 JavaScript를 이기고 대중화 될 날이 머지 않았다고 생각하여 해당 언어의 공부를 진행해보고 싶었다.</p><p>- 올해 초부터 웹 개발에 재미를 느껴 react, javaspring, node.js 등 여러 프레임워크들의 공부를 진행 해보았지만 파이썬의 웹개발 프레임워크인 Flask의 공부를 깊게 해본 적이 없어서 해당언어의 공부를 진행해보고 싶었다.</p><p>**1.3 프로젝트 목표**</p><p>- 한 학기 동안 프로그래밍응용 강의에서 배운 내용들을 해당 프로젝트의 정리하는 사이트를 개발</p><p>- Flask(파이썬)를 통해 해당 웹 구조를 설계</p><p>- 강의의 Weekly HomeWork의 문제와 입력코드, 출력결과를 파이스크립트를 이용해 사이트에 게시</p><p>- 파이썬 언어는 아니지만 해당 사이트 이용자들의 편의를 위해 bootstrap프레임워크를 활용해 디자인</p><p>- 파이썬 코드를 웹에서 파이스크립트(인터프리터)를 활용해 결과를 출력</p><p>- 해당 웹사이트에 MBTI검사 페이지를 개발</p><p>- 여름방학에 파이썬으로 진행했던 토이프로젝트(재고관리 프로그램)을 업데이트하여 추가</p>|
